# ADR-0001: Record Architecture Decisions

## Status
Accepted

## Context
CIXCI architecture decisions need a durable place to capture why platform-level choices were made.

## Decision
Use architecture decision records in this folder for significant cross-module platform decisions.

## Consequences
- Decisions can be reviewed and pressure tested over time.
- Future module work can reference prior platform decisions.
- Open questions should remain explicit until resolved.
