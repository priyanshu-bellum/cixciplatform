# System Boundaries

## CIXCI Owns
- Catalog normalization
- Visibility rules
- Product activation mappings
- Order routing
- Shipping and return orchestration
- Analytics and recommendation logic

## External Systems Own
- Vendor source systems
- Buyer storefronts
- Manufacturer source systems
- Payment processing
- Carrier/MVNO commerce platforms

## Boundary Questions
- What remains system-of-record vs synchronized data?
- What should be event-driven vs API request-driven?
